Reflection Questions

1. How does using alt text improve accessibility?

If an image is unable to load or there is an issue with the browser it still allows the user a way to see a description of the image so they can understand the context on the webpage


2. Why should you use multiple <source> elements for video and audio?

Like images, if a browser or user is unable to load a specific file type such as .wav then having a second or multiple identical files, differing only by file type, then it allows a better chance for the user to see the content.


3. What are the advantages of embedding a YouTube video instead of hosting it directly?

You do not have to host, store, serve, etc the file yourself, rather YouTube does. Additionally, youtube has many features built in such as quto quality configuration, cross-platform functionality, and many more that you would not have to implement.
